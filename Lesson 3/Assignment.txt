Hint: for doubleChar, you can add characters to a string this way:
	String myString = "";
	char myChar = 'A';
	
	myString = myString + myChar + myChar // Adds 2 'A's to myString

Note that using the plus-assignment operator (+=) will NOT work:

	myString += myChar + myChar // Will not work!
